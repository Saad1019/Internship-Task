Task 1: Exploring and Visualizing the Iris Dataset¶

Introduction and Problem Statement¶

This task is part of the Data Science & Analytics Internship at DevelopersHub Corporation. The goal is to explore the Iris Dataset, understand its structure, and visualize relationships between variables

Dataset Understanding and Description¶

The Iris Dataset contains 150 samples of iris flowers with 4 features:

● Sepal length (cm)

● Sepal width (cm)

● Petal length (cm)

● Petal width (cm) It also includes the species (setosa, versicolor, virginica). The dataset is clean with no missing values.

Data Cleaning and Preparation¶

No cleaning is needed as the dataset has no missing values or inconsistencies.

Exploratory Data Analysis (EDA)¶

We will create visualizations to understand relationships, distributions, and potential outliers.

# Scatter plot: Petal length vs. Petal width by species
plt.figure(figsize=(8, 6))
sns.scatterplot(data=iris, x='petal_length', y='petal_width', hue='species', style='species')
plt.title('Petal Length vs. Petal Width by Species')
plt.xlabel('Petal Length (cm)')
plt.ylabel('Petal Width (cm)')
plt.show()

Conclusion¶

● The scatter plot shows that petal length and width are strongly correlated, with clear separation between species (e.g., setosa has smaller petals).

● Histograms reveal that petal length and width have multimodal distributions, indicating distinct groups (species).

● Box plots show minimal outliers, with sepal width having slightly more variability than other features.

● The Iris Dataset is well-structured and ideal for learning data exploration and visualization techniques.
